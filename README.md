**AEL Digital Studio** — Ayman Elmasry

## What is this?

AEL Engineering Academy is a **specification-driven learning platform** for engineering education. Built on top of the AEL Reference Framework, it provides structured learning paths with exercises, challenges, quizzes, and certification.

## v1.0.0 — LLM Engineering

The first learning path: **LLM Engineering** — from zero to production.

### Curriculum

| Module | Weeks | Topics |
|--------|-------|--------|
| Foundations | 1-3 | What are LLMs, Transformer Architecture, Tokenization |
| Core Techniques | 4-6 | Prompt Engineering, Fine-tuning, RAG Architecture |
| Applied Engineering | 7-9 | Agents & Tools, Evaluation, Deployment |
| Production & Capstone | 10-12 | Safety & Alignment, Advanced Topics, Capstone Project |

### Learning Model

```
Learn → Practice → Build → Validate → Certify → Portfolio
```

### Features

- 12 weeks of structured content
- 36 exercises (basic, intermediate, advanced)
- 12 weekly challenges
- 12 quizzes with scoring
- 60+ interview questions
- Progress tracking (localStorage)
- Search across all content
- Export (PDF, Markdown, JSON)
- Responsive design (desktop, tablet, mobile)
- Dark mode with glassmorphism UI

## Quick Start

```bash
# Clone
git clone https://github.com/aymanelmasryael/ael-engineering-academy.git
cd ael-engineering-academy

# Open
open index.html
```

No build step. No dependencies. Just HTML, CSS, and JavaScript.

## Architecture

Built with the same design language as:
- AEL LLM Engineering Reference
- AEL Terminal Engineering Reference
- AEL Reference Framework

## Tech Stack

- HTML5
- CSS3 (Custom Properties, Glassmorphism)
- Vanilla JavaScript (ES6+)
- No frameworks
- No external dependencies

## Progress Persistence

All progress is saved to `localStorage`. Your learning state persists across browser sessions.
